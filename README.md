# graphoneme_aligner
grapheme-phoneme alignment for two corpora


выравнивание и подсчеты метрик выполняются с помощью автоматическое_выравнивание.ipynb

в /intermediate tables/ содержатся примеры сопотавительных таблиц для обоих корпусов

в папках /results .../ содержатся таблицы с результатами некоторых вычислениях
